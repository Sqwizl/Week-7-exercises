5. Simple Password Checker

Write a program that checks if a password (stored in `password`) matches a predefined password (stored in `expectedPassword`, could be `thisismypassword`) and displays a message like "Access Granted" or "Access Denied."

Hint: Use the `===` operator to compare the two passwords.


let password = 'thisismypassword'; let expectedpassword = 'thisismypassword'; if (password === expectedPassword) { console.log('Access Granted') } else { console.log ('Access Denied') }
