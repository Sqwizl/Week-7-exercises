11. Weight Converter

Write two functions: one to convert imperial pounds to kilograms and another to convert kilograms to imperial pounds. Each function should take one parameter (the weight) and return the converted value. The formula for that is: kilograms = pounds / 2.2 and pounds = kilograms * 2.2
Example call:console.log(kilosToPounds(10)); // should log `22`

Convert all these values: 12 pounds 14 pounds 0 pounds -5 pounds 1 kilo 0.512 kilos 2342324323 kilos
