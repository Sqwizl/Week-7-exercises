8. Math Wizard



Create a function for each of the following operations: addition, subtraction, multiplication, and division. The functions should take two numbers as parameters and return the result of the operation.

Example:
function add(num1, num2) { return num1 + num2;} console.log(add(3, 5)); // 8
function subtract(num1, num2) { return num1 - num2;} console.log(subtract(3, 5)); // -2
function multiply(num1, num2) { return num1 * num2;} console.log(multiply(3, 5)); // 15
function divide(num1, num2) { return num1 / num2;} console.log(divide(3, 5)); // 0.6
