4. Simple Grading System

Write a program that takes a student's test score (e.g., 85) and prints out their grade (e.g., "A" for scores 90 and above, "B" for 80-89, etc.). Anything below score 50 is an "F". Use if/else statements.

let score = 85; if (score >= 90) { console.log("A"); } else if (score >= 80) { console.log("B"); } else if (score >= 70) { console.log("C"); } else if (score >= 60) { console.log("D"); } else if (score <= 50) { console.log("F"); }
