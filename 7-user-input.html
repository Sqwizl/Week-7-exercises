7. How to ask the user for input

7a.Find out how to use the prompt() function in JavaScript.

The prompt() function is used to display a window dialog box. It displays a prompt box that asks the user to input value. The prompt() function returns the value entered by the user. Otherwise NULL is returned.
Syntax would be let user = prompt ("message", "default value");


Example:
let person = prompt("what is your name?", "John Doe") console.log(person);
